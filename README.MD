# Student Performance Prediction (Simple ML Project)

This is a small, beginner-friendly machine learning project that predicts a student's marks from simple study-related features. I built this to learn the whole ML workflow: load data → train a model → evaluate → get predictions.

---

## Files in this repository

- `main.py` — The program you run. It loads data, trains a linear regression model, and prints sample predictions.
- `model.py` — Contains the model training function.
- `student_performance.csv` — The dataset (100 realistic rows). Features: Hours_Studied, Attendance, Sleep_Hours, Previous_Score, Marks.
- `requirements.txt` — Python packages required.

---

## How to run (Windows / VS Code)

1. Open this folder in VS Code.
2. Open the terminal (View → Terminal).
3. Install dependencies (only once):

pip install -r requirements.txt
